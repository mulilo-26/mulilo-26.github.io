## 📌 Project Highlights

### Layoffs Data Cleaning
Cleaned a raw dataset of global tech layoffs including:
- Removing duplicates using `ROW_NUMBER()` and CTEs
- Standardising inconsistent text values and date formats
- Populating NULL values using self-joins
- Dropping irrelevant columns

### Employee Database
Built and queried a relational employee database including:
- Designing triggers to sync data across tables on insert
- Writing complex joins to analyse employee data
- Handling NULL values and edge cases
